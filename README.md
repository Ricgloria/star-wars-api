# MS star-wars-api

Serviço Java responsável pela gestão de filmes Star Wars.

### Pré-requisitos

- Java JDK 17
- Maven 3.8+

## Comandos Maven

- Compilar project

  `mvn compile`

- Limpar diretorio Target

  `mvn clean`

- Executar testes do projeto

  `mvn test`

- Executar projeto

  `mvn spring-boot:run`

- Instalar pacotes

  `mvn install`
